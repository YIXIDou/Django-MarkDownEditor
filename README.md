## 📖 项目简介
这是一个基于 **Django 5.x** 框架开发的全栈 Web 应用。旨在解决现有博客平台不支持 **Markdown** 原生语法的痛点，为开发者提供一个“所见即所得”的沉浸式创作环境。

## ✨ 核心亮点
- **原生 Markdown 支持**：集成 `django-markdownx`，支持代码高亮、公式渲染与实时预览。
- **沉浸式双栏设计**：自定义 CSS Grid 布局，左侧编辑右侧预览，还原专业 IDE 体验。
- **全栈架构**：基于经典的 **Django MTV** 模式，实现了完整的用户鉴权与 CRUD 流程。
- **响应式 UI**：基于 Bootstrap 4，完美适配移动端与 PC 端。

## 🛠️ 技术栈
- **Backend**: Python 3.12, Django 5.0
- **Frontend**: HTML5, CSS3, JavaScript, Bootstrap 4
- **Database**: SQLite (Dev)
- **Tools**: Git, VS Code
